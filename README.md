## project Description

This project focuses on building a CLI application that retrieves vehicle recall data from public APIs. Users can search recalls by vehicle model, manufacturer, or year using command-line inputs. The system should store recall data, prevent duplicates, and support filtering. This project is conceptually very similar to the CVE system but applied to the automotive domain.
